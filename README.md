# 돌리와 도트

#### 부산 학생주도형 제 4회 메이커랑 놀자



## 참가 학생

대표 : 용호중학교 3721 최재민
팀원 : 3310 송찬혁, 3325 조용현





## 작품 설명

### 사용한 재료

방수 스프레이, 소켓 점퍼 MM, MF, FF, 외경 7mm 스프레이 노즐, 내경 7mm 실리콘 튜브, 워터펌프 모터, 아두이노 우노용 USB A to B 케이블, 브레드 보드, Arduino Uno R3 SMD 호환보드, 마이크, 노트북, 에탄올 1L, 소독제 담을 넓고 높은 용기
